## 🚀 项目愿景

**打破信息差，让每个人都能轻松驾驭 AI 智能体。**

OpenClaw 是目前最热门的开源 AI 助手平台之一，但高质量的中文资源、实战案例和最佳实践往往分散在各个角落，对于初学者和开发者来说，不仅难以寻找，还容易走弯路。

**Botguide** 的使命是构建一个**全面、专业、实时更新**的 OpenClaw 知识库和资源中心。我们不仅聚合了全网最优质的教程和文档，更致力于挖掘真实的应用场景和解决方案，帮助你：

*   🔍 **快速找到所需**：无需大海捞针，一站式获取所有关键信息。
*   💡 **获得灵感启发**：通过丰富的案例，发现 OpenClaw 的无限可能。
*   🛠️ **少走技术弯路**：基于社区验证的最佳实践，直接复用成熟方案。
*   🤝 **连接开发者**：加入活跃的社区，与志同道合的朋友共同成长。

## ✨ 核心特性

*   **现代化设计**：基于 Next.js 15 + Tailwind CSS 4.0 构建，提供流畅、美观的浏览体验。
*   **精选资源**：人工筛选并分类的优质教程、视频、文章和工具。
*   **实战导向**：不仅有理论，更强调“怎么用”，提供可落地的场景案例。
*   **完全开源**：项目代码和数据完全开放，欢迎任何人参与贡献。

## 📦 快速开始

### 环境要求

*   Node.js 18+
*   npm / yarn / pnpm

### 安装与运行

1.  **克隆仓库**

    ```bash
    git clone https://github.com/yourusername/botguide.git
    cd botguide
    ```

2.  **安装依赖**

    ```bash
    npm install
    # 或者
    yarn install
    # 或者
    pnpm install
    ```

3.  **启动开发服务器**

    ```bash
    npm run dev
    ```

    打开浏览器访问 [http://localhost:3000](http://localhost:3000) 即可看到效果。
